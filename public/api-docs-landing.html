APIs de Conversion

POST /api/convert
Convertit un message HL7 v2.5 en ressources FHIR R4 (format JSON)
// Requête POST /api/convert X-API-KEY: dev-key Content-Type: application/json { "hl7Message": "MSH|^~\\&|SENDING_APP|SENDING_FACILITY|..." }
POST /api/convert/raw
Convertit un message HL7 v2.5 en ressources FHIR R4 (format texte brut)
// Requête POST /api/convert/raw X-API-KEY: dev-key Content-Type: text/plain MSH|^~\\&|SENDING_APP|SENDING_FACILITY|...
POST /api/convert/mllp
Convertit un message HL7 v2.5 encapsulé dans le protocole MLLP en ressources FHIR R4
// Requête POST /api/convert/mllp X-API-KEY: dev-key Content-Type: application/mllp [caractères MLLP]MSH|^~\\&|SENDING_APP|...[caractères MLLP]

APIs Workflows

GET /api/workflows
Liste tous les workflows configurés
// Requête GET /api/workflows X-API-KEY: dev-key
GET /api/workflows/:id
Récupère les détails d'un workflow spécifique
// Requête GET /api/workflows/1 X-API-KEY: dev-key
POST /api/workflows
Crée un nouveau workflow
// Requête POST /api/workflows X-API-KEY: dev-key Content-Type: application/json { "name": "Nouveau Workflow", "description": "Description du workflow", "application_id": 1, "flow_json": { "nodes": [], "edges": [] } }
PUT /api/workflows/:id
Met à jour un workflow existant
// Requête PUT /api/workflows/1 X-API-KEY: dev-key Content-Type: application/json { "name": "Workflow Modifié", "flow_json": { "nodes": [], "edges": [] } }
DELETE /api/workflows/:id
Supprime un workflow
// Requête DELETE /api/workflows/2 X-API-KEY: dev-key
POST /api/workflows/execute
Exécute un workflow à partir de sa définition dans le corps de la requête
// Requête POST /api/workflows/execute X-API-KEY: dev-key Content-Type: application/json { "flow": { "nodes": [], "edges": [] }, "input": "MSH|^~\\&|SENDING_APP|..." }
POST /api/workflows/:id/execute
Exécute un workflow existant avec les données d'entrée fournies
// Requête POST /api/workflows/1/execute X-API-KEY: dev-key Content-Type: application/json { "input": "MSH|^~\\&|SENDING_APP|..." }
GET /api/workflows/:id/export-template
Exporte un workflow sous forme de template réutilisable
// Requête GET /api/workflows/1/export-template X-API-KEY: dev-key
POST /api/workflows/import-template
Importe un template de workflow
// Requête POST /api/workflows/import-template X-API-KEY: dev-key Content-Type: application/json { "template": { /* template JSON */ }, "application_id": 1, "name": "Workflow depuis template" }
