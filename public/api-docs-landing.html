Documentation API FHIRHub

Documentation complète de l'API pour le convertisseur HL7 vers FHIR

Informations d'authentification

Clé API de test: dev-key

Endpoint de base: /api

Header d'authentification: X-API-KEY: dev-key

APIs de Conversion

POST /api/convert
Convertit un message HL7 v2.5 en ressources FHIR R4 (format JSON)
// Requête POST /api/convert X-API-KEY: dev-key Content-Type: application/json { "hl7Message": "MSH|^~\\&|SENDING_APP|SENDING_FACILITY|..." }
POST /api/convert/raw
Convertit un message HL7 v2.5 en ressources FHIR R4 (format texte brut)
// Requête POST /api/convert/raw X-API-KEY: dev-key Content-Type: text/plain MSH|^~\\&|SENDING_APP|SENDING_FACILITY|...
POST /api/convert/mllp
Convertit un message HL7 v2.5 encapsulé dans le protocole MLLP en ressources FHIR R4
// Requête POST /api/convert/mllp X-API-KEY: dev-key Content-Type: application/mllp [caractères MLLP]MSH|^~\\&|SENDING_APP|...[caractères MLLP]

APIs de Terminologies

GET /api/terminology/french
Récupère les informations sur les terminologies françaises utilisées pour la conversion
// Requête GET /api/terminology/french X-API-KEY: dev-key
GET /api/terminology/systems
Liste tous les systèmes de terminologie disponibles
// Requête GET /api/terminology/systems X-API-KEY: dev-key

APIs Système

GET /api/stats
Récupère les statistiques du système (conversions, performances)
// Requête GET /api/stats X-API-KEY: dev-key
GET /api/applications
Liste toutes les applications enregistrées dans le système
// Requête GET /api/applications X-API-KEY: dev-key
GET /api/api-keys
Liste toutes les clés API disponibles
// Requête GET /api/api-keys X-API-KEY: dev-key
GET /api/workflows
Liste tous les workflows configurés
// Requête GET /api/workflows X-API-KEY: dev-key
Note de sécurité: Toutes les API requièrent une authentification via une clé API. Assurez-vous de protéger vos clés API et de ne pas les partager.
