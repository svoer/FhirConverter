GET
​/fhir​/CodeSystem
Obtenir tous les CodeSystems présents dans le point de terminaison FHIR.

Cette API permet d'obtenir tous les CodeSystems présents. Si vous voulez accéder aux CodeSystems protégés, s’identifier en utilisant le header HTTP "Authorization" avec une valeur de type Bearer Token ("Bearer xxxxxxx...")

Parameters
No parameters

Responses
Code	Description
200	
L'ensemble des CodeSystems si la requête aboutit avec succès.

Media type

application/fhir+json
Controls Accept header.
Example Value
Schema
[
  {
    "resourceType": "Bundle",
    "total": "string"
  }
]
401	
Jeton non valide.

403	
L'utilisateur ne possède pas les droits pour effectuer cette requête.

404	
Une erreur dans la requête.

500	
Erreur interne du serveur.
