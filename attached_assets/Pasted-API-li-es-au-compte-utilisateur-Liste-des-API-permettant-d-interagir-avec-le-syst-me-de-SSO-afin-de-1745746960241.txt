API liées au compte utilisateur
Liste des API permettant d'interagir avec le système de SSO afin de d'obtenir des jetons d'authentification pour appeler les API nécessitant une authentification


POST
​/ans​/sso​/auth​/realms​/ANS​/protocol​/openid-connect​/token
Se connecter à son compte utilisateur ou rafraîchir sa session
POST
​/ans​/sso​/auth​/realms​/ANS​/protocol​/openid-connect​/logout
Déconnecter sa session utilisateur
API pour les terminologies
Liste des API permettant d'interagir avec les terminologies.


GET
​/wp-json​/ans​/terminologies​/zip
Télécharger le ZIP d'une terminologie.

GET
​/wp-json​/ans​/terminologies​/versions-details
Obtenir les versions disponibles d'une terminologie.

GET
​/api​/terminologies​/search
Obtenir la liste des terminologies présentes dans le catalogue des terminologies.

GET
​/api​/terminologies​/search​/autocomplete
Obtenir les résultats d'autocomplétion d'un champ.

GET
​/api​/terminologies​/search​/advanced
Obtenir la liste des terminologies répondant aux critères d'une recherche avancée.

GET
​/api​/terminologies​/list
Obtenir la liste des terminologies visibles.

GET
​/api​/terminologies​/list​/with-concepts
Obtenir la liste des terminologies visibles ayant des concepts.

GET
​/api​/terminologies​/home
Obtenir la liste des terminologies présentes en page d'accueil.

GET
​/api​/terminologies​/detail
Obtenir le détail d'une terminologie.

GET

GET

API liées au serveur FHIR
Liste des API permettant d'intéragir avec le point de terminaison FHIR.


GET
​/fhir​/CodeSystem
Obtenir tous les CodeSystems présents dans le point de terminaison FHIR.

GET
​/fhir​/CodeSystem​/{id}
Obtenir un CodeSystem par son id.

GET
​/fhir​/CodeSystem​/{id}​/_history​/{version_id}
Obtenir une version d'un CodeSystem par son id et sa version.

GET
​/fhir​/CodeSystem​/{id}​/_history
Obtenir l'historique d'un CodeSystem par son id.

GET
​/fhir​/CodeSystem​/_search
Permet d'effectuer une recherche sur les CodeSystems présents sur le serveur.

GET
​/fhir​/CodeSystem​/$lookup
Obtenir toutes les informations sur un code.

GET
​/fhir​/CodeSystem​/{id}​/$validate-code
Savoir si un code fait partie d'un CodeSystem.

GET
​/fhir​/CodeSystem​/$subsumes
Connaitre le niveau de hiérarchie entre deux codes ou codings.

GET
​/fhir​/ValueSet
Lister tous les ValueSets.

GET
​/fhir​/ValueSet​/{id}
Obtenir un ValueSet par son id.

GET
​/fhir​/ValueSet​/{id}​/_history​/{version_id}
Obtenir une version d'un ValueSet par son id et sa version.

GET
​/fhir​/ValueSet​/{id}​/_history
Obtenir l'historique d'un ValueSet par son id.

GET
​/fhir​/ValueSet​/_search
Permet d'effectuer une recherche sur les ValueSets présents sur le serveur.

GET
​/fhir​/ValueSet​/$expand
Etendre le contenu d'un ValueSet.

GET
​/fhir​/ValueSet​/{id}​/$validate-code
Savoir si un code fait partie d'un ValueSet.

GET
​/fhir​/ConceptMap
Lister tous les ConceptMaps.

GET
​/fhir​/ConceptMap​/{id}
Obtenir un ConceptMap par son id.

GET
​/fhir​/ConceptMap​/{id}​/_history​/{version_id}

GET
​/fhir​/ConceptMap​/{id}​/_history

GET
